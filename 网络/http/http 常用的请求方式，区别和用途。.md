### http 常用的请求方式，区别和用途。

> HTTP 常用的请求方式、区别和用途

| 请求方式 | 用途                                                         |
| :------- | :----------------------------------------------------------- |
| get      | 对服务器资源获取的简单请求                                   |
| post     | 向指定资源提交数据进行处理请求（例如提交表单或者上传文件）。 数据包含在请求体中。 POST 请求可能会导致新的资源的建立和/或已有资源的修改。 |
| put      | 向服务器提交数据，以修改数据                                 |
| delete   | 删除服务器上的某些资源                                       |
| patch    | 是对 put 方法的补充，用来对某个资源做局部修改                |
| head     | 请求页面的首部，获取资源的元信息                             |
| connect  | 用于 ssl 隧道的基于代理的请求                                |
| options  | 允许客户端查看服务器的性能                                   |
| trace    | 回显服务器收到的请求，主要用于测试或诊断                     |
